# contact

이 프로젝트는 COPD(만성 폐쇄성 폐질환) 환자들을 위한 의료용 기능성 게임의 시작점입니다. 게임은 사용자가 풍선을 불어 터트리는 방식으로 진행되며, 이 과정에서 환자들이 호흡 훈련을 자연스럽게 할 수 있도록 설계되었습니다. 게임 내에서 숨을 오랫동안 불기 위해서는 현재 지역의 날씨 데이터를 활용하여 바람의 세기나 날씨 조건에 따라 게임의 난이도를 조절합니다.
